# ResNet
https://arxiv.org/pdf/1512.03385.pdf

## Dataset

- CIFAR10

## Hyperparameter Setting

- **Batch Size**: 256
- **Optimizer**: SGD
- **Learning Rate**: 0.001 (momentum=0.9, weight_decay=0.0001)
- **Num Epochs**: 200
- **Loss Function**: CrossEntropyLoss

## Results

- **Train Time**:  1,850 seconds
- **Accuracy**: 74.85%
